# java8 Chef Cookbook

##Purpose##

This chef cookbook installs and configures Oracle Java 8 on the target machine.

Requirements
------------
### Platforms
- Debian, Ubuntu (OpenJDK, Oracle)
- CentOS 6+, Red Hat 6+, Fedora, Amaxon (OpenJDK, Oracle), Scientific Linux 6

### Dependencies
- java

Usage
-----
Simply include the recipe ```java8::default``` in your runlist.
